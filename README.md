🌐 Reyhan Portfolio Website

Website portfolio modern, responsif, dan interaktif yang menampilkan profil, project, serta fitur komentar realtime.

---

✨ Fitur Utama

- 🎬 Intro animasi (smooth & modern)
- 🎨 UI/UX minimalis dengan warna neon (ungu, pink, biru)
- 📱 Responsive design (mobile & desktop)
- 🎵 Favorite Music section (custom player)
- 💬 Sistem komentar realtime
- ⚡ Animasi scroll (smooth & repeatable)
- 🌌 Background animasi (aurora neon)

---

🛠️ Teknologi yang Digunakan

- React (Vite)
- TailwindCSS
- JavaScript / TypeScript
- Supabase (Realtime Database)

---

🚀 Fitur Komentar Realtime

Website ini menggunakan Supabase untuk:

- Menyimpan komentar user
- Menampilkan komentar secara live tanpa refresh
- Sinkronisasi data secara otomatis

---

⚙️ Setup Project

1. Clone repository:

git clone https://github.com/username/repo-name.git
cd repo-name

2. Install dependency:

pnpm install

3. Jalankan project:

pnpm run dev

---

🔐 Environment Variables

Buat file ".env" lalu isi:

VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_anon_key
